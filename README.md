In this project, I will be analyzing the Emporium Bookstore Sales data.
I will be creating the following visuals;
- A line chart showing sales trends over the full period covered in the data
- A column or bar chart showing sales performance by category
- A pie or donut chart showing relative sales for each state in the region
- A list of top-selling books, including author names, for your region (general audience, excluding textbooks)
